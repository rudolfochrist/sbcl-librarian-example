
# sbcl-librarian-example

Dabblings with [sbcl-librarian](https://github.com/quil-lang/sbcl-librarian).

## Install

Checkout code, then:

```sh
cd lib
make
make test
make check
```

## Create new lib

Look at [sbcl-librarian.project](https://github.com/quil-lang/sbcl-librarian/blob/main/src/project/README.md).
